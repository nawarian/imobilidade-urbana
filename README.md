Imobilidade urbana (remake)
---

Remake to jogo [Imobilidade Urbana](https://ntmtst.itch.io/imobilidade-urbana) lançado em
Janeiro de 2024.

Esta versão utiliza a [Godot engine](https://godotengine.org/) e traz algumas otimizações.
